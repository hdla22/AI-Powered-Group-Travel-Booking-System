# AeroHive – AI-Powered Group Travel Booking System 

AeroHive is an intelligent group travel booking solution built for the **Riyadh Air Hackathon 2025**.  
It solves the complex problem of **coordinating group travel decisions and bookings** by combining:

- Smart group voting
- Automated booking execution using AI Agents
- Android application interface
- Real-time notifications and group communication

---

## Problem Statement 

Group travelers face major difficulties when trying to:

- Agree on travel dates and times  
- Coordinate preferences manually through chat  
- Handle group payments and confirmations  
- Avoid errors in multi-passenger bookings  

These challenges make group booking slow, messy, and error-prone.

---

## Our Solution – AeroHive 

AeroHive automates the entire **group travel decision and booking flow** through:

1. **Group Voting System**
   - Members vote on preferred dates and times
   - Majority-based decision selection
   - Tie-handling logic supported

2. **AI Booking Agent**
   - Automatically executes group booking
   - Generates structured booking confirmation
   - Calculates total cost
   - Generates individual tickets for each passenger

3. **Android Application**
   - Group travel chat
   - Survey & voting interface
   - Booking confirmation display
   - Loyalty points & wallet system

4. **Smart Notifications**
   - Progress-based booking updates
   - Real-time alerts and confirmations

---

## Use of AI 

- AI Agent executes the **entire booking logic automatically**
- Generates:
  - Booking confirmation
  - Passenger records
  - Ticket numbers
  - Pricing calculations
- Eliminates all manual backend processing
